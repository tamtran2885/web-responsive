# Responsive Web

This project was generated with HTML, CSS and SASS. This work is based on the idea to create a responsive web app which later can be adapted to mobile version.

## Technologies used

- HTML
- CSS
- SASS
- NodeJS

## Installation

Install dependencies.

```http
 npm install
```

## Development

```http
 npm run watch
```

## Build

```http
 npm run build
```
